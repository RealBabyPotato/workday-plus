# components

this folder is for reusable ui pieces.

what goes here:

- buttons
- cards
- forms
- modals
- schedule blocks

hello-world file:

- `SectionCard.tsx` is a tiny reusable ui component

why this exists:

- shared components reduce duplicate code
- it keeps styling and behavior consistent

install:

- no separate install
- install the frontend app from [apps/web](../../README.md)

launch:

- no separate launch
- component files render inside the frontend app
