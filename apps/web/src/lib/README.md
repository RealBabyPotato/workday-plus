# lib

this folder is for small shared frontend helpers.

what goes here:

- utility functions
- constants
- formatting helpers
- small shared hooks later if needed

hello-world file:

- `demo-data.ts` shows a tiny shared helper/data file

why this exists:

- it prevents helper code from ending up in random places
- it keeps features focused on product logic

install:

- no separate install
- install the frontend app from [apps/web](../../README.md)

launch:

- no separate launch
- helper files are imported by frontend features when needed
