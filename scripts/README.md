# scripts

this folder is for one-off project scripts.

what goes here:

- local setup helpers
- seed scripts
- data import scripts

why this exists:

- scripts are useful, but they should not be hidden inside app code
- this makes operational tasks easier to find

install:

- no separate install
- run scripts from their own folders once they exist

launch:

- scripts are run directly when they are added
- there is no single launcher for this folder
